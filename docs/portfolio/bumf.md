# BumF Website Relaunch <!-- {docsify-ignore-all} -->

I created the initial design for the website relaunch of the [Bundesfachverband für minderjährige unbegleitete Flüchtlinge (BumF)](https://b-umf.de/ ':target=_blank'). The design focuses on two things in particular. First, the website must provide a lot of information and be easy to navigate. Second, it should encourage visitors to donate or subscribe to the newsletter or RSS feed.

Freelance work for [part](https://part.berlin/ ':target=_blank'), 2017.
