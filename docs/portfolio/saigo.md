# SAIGO Font <!-- {docsify-ignore-all} -->

Some of the typography scribbles I made during long meetings turned out to be more interesting than others, so I decided to put time and effort into bringing one of them to life as the font SAIGO, which is the Japanese word for "end." The font is only suitable for headlines and currently has a limited character set. If you'd like to help continue this project, send me an email.

Personal project, 2015.
