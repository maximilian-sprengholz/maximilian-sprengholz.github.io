# JPBW Anniversary Book <!-- {docsify-ignore-all} -->

This book celebrates the 30th anniversary of the [Jugendpresse BW](https://jpbw.de/ ':target=_blank') (JPBW, a student press association) and showcases the new [corporate design](./portfolio/jpbw_cd.md) I created for them. It is a 64-page B5-format book containing information about the history and current activities of JPBW, as well as interviews with former and current members. The book was presented at Jugendmedientage 2013 in Karlsruhe and is now used as an image brochure. If not for the tight budget, the cover would have been punched in the shape of the 30, revealing the colorful logo from the first page.

Freelance work, 2013.
