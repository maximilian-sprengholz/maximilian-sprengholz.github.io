# SWR1 CLP Kampagne <!-- {docsify-ignore-all} -->

Für ein Leuchtplakat-Design für den deutschen Radiosender SWR1 haben wir echte Schallplatten anhand von Vektordaten lasergraviert und anschließend fotografiert. Laut SWR1 hatte der Sender noch nie zuvor so viel positives Feedback von Hörer*innen zu einer Kampagne erhalten.

Design erstellt bei [fischerAppelt Stuttgart](http://www.fischerappelt.de ':target=_blank') mit [Davide Di Donna](http://be.net/DavideDiDonna ':target=_blank') und [Jedrzej Golecki](https://www.behance.net/jtgolecki9ae3 ':target=_blank'), 2013.
