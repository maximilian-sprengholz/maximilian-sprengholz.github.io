# SAIGO Schrift <!-- {docsify-ignore-all} -->

Eine meiner vielen typografischen Skizzen aus langen Besprechungen fand ich spannend genug, um die Zeit und Mühe zu investieren, sie zum digitalen Leben zu erwecken. So entstand die Schriftart SAIGO, was auf Japanisch "Ende" bedeutet. Die Schriftart eignet sich nur für Überschriften und verfügt derzeit über einen begrenzten Zeichensatz. Wenn du Lust hast, dieses Projekt fortzusetzen, schreibe mir gern eine E-Mail.

Persönliches Projekt, 2015.
